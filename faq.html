Subarashii no Fansub

Qu'est-ce que la SnF ?

La SnF, pour Subarashii no Fansub, est une petite fansub, une petite équipe de traduction ou encore une équipe de release.

Quel est le format de vos vidéos ?

C'est du .mkv, souvent 720p car c'est la définition qui, selon moi, propose un bon ratio qualité/poids. Bien sûr, dans la mesure du possible, ça sera du 10bits. Ceci est valide pour les releases TV et BD.

Les logiciels de lecture que nous conseillons :

  • Pour Windows : MPC-HC
  • Pour Mac OS & Linux : MPV
  • Pour les autres (Android, iOS, …) : Kodi, MXPlayer, VLC ?, Infuse 3 ?

Les logiciels de torrent que nous conseillons :

  • qBitorrent

À quoi sert le deuxième code entre crochets ?

Ce code est appelé contrôle de redondance cyclique (CRC32).
Elle permet de vérifier si le fichier est complet (et n'a pas été corrompu durant le téléchargement).
Le torrent permet automatiquement de vérifier cela, mais pas les autres types de téléchargement.
Nous conseillons :
  • Pour Windows : RapidCRC, RapidCRC-Unicode
  • Pour Mac : de base dans votre distribution, en ligne de commande,
  • Pour Linux : voir ce wiki

Comment patcher un épisode ?

Il arrive que des coquilles s’introduisent dans le script des sous-titres, qu’une faute traîne ou qu’un terme ne soit pas adéquat. Il est alors d’usage de proposer une « v2 » de l’épisode ou un batch.
Dans certains cas, nous fournissons uniquement le sous-titre .ass, à vous de le muxer avec l'épisode. Dans d'autres cas, nous fournissons un fichier .zip qui contient un .bat, .sh et .vcdiff :
  • Pour Windows : dans le même dossier que l'épisode à patcher, double cliquez sur le fichier ".bat".
  • Pour Linux : installer xdelta3, et éxecutez le fichier .sh dans le même dossier que l'épisode à patcher
  • Pour Mac : installer Homebrew et écrivez brew install xdelta dans votre terminal pour installer xdelta ; écrivez ensuite dans le même dossier que l'épisode à patcher xdelta3 -d -s ancien_mkv.mkv Patch.vcdiff nouveau_mkv.mkv
Vous verrez apparaître un second fichier en V2. Il ne vous reste désormais plus qu'à supprimer tous les fichiers pour ne garder que l'épisode V2 (que vous pouvez renommer si vous le souhaitez) !
